A 2D Metal Slug-inspired game developed as an Object-Oriented Programming project using C++. The project demonstrates core OOP concepts including Composition, Aggregation, Inheritance, and Polymorphism.

Features include:

* Smooth sprite-sheet based animations
* Procedural terrain generation using Perlin Noise
* Infinite level generation
* Enemy AI and vehicle systems
* Efficient class hierarchy design with solutions to complex inheritance problems such as the Diamond Problem using pure virtual functions

This project was a great learning experience in game development, problem solving, and advanced OOP design principles.
